Установка

Четыре шага
до запуска

Весь процесс занимает меньше минуты. Никаких лишних программ.

1

Скачайте файл

Нажмите кнопку «Скачать» — файл RodinaHelper.lua сохранится на компьютер.

2

Откройте папку игры

Перейдите в корневую директорию «Родина Онлайн» — туда, где лежит исполняемый файл.

3

Скопируйте в MoonLoader

Найдите папку MoonLoader/ и скопируйте туда скачанный файл.

Родина Онлайн / MoonLoader / RodinaHelper.lua
4

Запустите игру

MoonLoader подхватит скрипт автоматически. Никаких дополнительных настроек не нужно.

moonloader — bash
$ls MoonLoader/
moonloader.dll   scripts/
$cp RodinaHelper.lua MoonLoader/
✓ Файл скопирован
$ls MoonLoader/
moonloader.dll   scripts/   RodinaHelper.lua
$./RodinaOnline.exe
✓ MoonLoader: RodinaHelper loaded
✓ Версия 3.6 активна
$
